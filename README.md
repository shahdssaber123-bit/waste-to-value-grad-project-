# ♻️ WasteToValue – Graduation Enterprise Project

## 📌 Overview

**WasteToValue** is an enterprise-level graduation project designed to transform waste management into a smart, value-driven system.

The platform connects users, businesses, and recycling operations in one ecosystem to:

* Reduce environmental waste 🌍
* Track recyclable materials ♻️
* Convert waste into economic value 💰
* Enable smart data-driven sustainability decisions 📊

---

## 🚀 Project Features

* Smart waste classification system
* Recycling workflow management
* User & organization dashboards
* Real-time tracking of waste-to-value operations
* Analytics & reporting system
* Enterprise-grade architecture
* Responsive UI for all devices

## 💻 Run the Project Locally

### 📦 Prerequisites

Make sure you have:

* Node.js installed
* npm or yarn
* Git installed

---

### 1️⃣ Clone the repository

```bash
git clone https://github.com/shahdssaber123-bit/waste-to-value-grad-project-.git
```

---

### 2️⃣ Navigate to project directory

```bash
cd waste-to-value-grad-project-
```

---

### 3️⃣ Install dependencies

```bash
npm install
```

---

### 4️⃣ Setup environment variables

Create a file named:

```bash
.env.local
```

Add the following:

```env
VITE_BASE44_APP_ID=your_app_id
VITE_BASE44_APP_BASE_URL=your_backend_url
```

### Example:

```env
VITE_BASE44_APP_ID=cbef744a8545c389ef439ea6
VITE_BASE44_APP_BASE_URL=https://your-backend.base44.app
```

---

### 5️⃣ Run the project

```bash
npm run dev
```
